PhysicsFS is a library 


PhysicsFS is a library to provide abstract access to various archives, which means that with this library we can access various archive formats regardless of their specific details. This makes for easy access to ZIP files in the same way as you access a file directly on the disk, and it makes it easy to ship a new archive that will override a previous archive on a per-file basis.
